Fall Jump

I developed this game for my high school capstone project in the fall of 2023. To make the game I used the GameMaker game engine which uses the GameMaker Language (GML). I also used a website called Itch.io to be able to publish the game online for anyone to play for free.

Fall Jump is a platformer game where you try to climb to the top as fast as you can. But this isn't as easy as it seems, you need to utilize strategic movements, teleportation and climbing through blocks all while avoiding falling back down and having to restart. At the end there is a local leaderboard to keep track of the fastest times on your machine.
